### 🌟 Proje Vizyonu
**ScholarGraph**, akademik dünyadaki bilgi kirliliğini aşmak ve makaleler arasındaki **"gizli genetik bağları"** keşfetmek için tasarlanmış yeni nesil bir analiz platformudur. Sadece kelimeleri değil, akademik literatürün **sinir ağlarını (atıfları)** haritalandırır.

---

## 🚀 Öne Çıkan "High-Tech" Özellikler

- **🧬 Dinamik Bilgi Grafiği (2D/3D):** Makaleler arasındaki karmaşık atıf ilişkilerini, parçacık efektli canlı bir ağ üzerinde görselleştirir.
- **⚡ Hibrit Veri Mimarisi:** MongoDB'nin esnek doküman yapısı ile Neo4j'nin ultra hızlı grafik ilişki motorunu birleştirir.
- **🤖 Akıllı Öneri Algoritması:** Neo4j Graph algoritmalarını kullanarak, konvansiyonel sistemlerin kaçırdığı "derinlemesine alakalı" içerikleri saniyeler içinde bulur.
- **💎 Premium Dashboard:** Tamamen Glassmorphism odaklı, modern, akıcı ve kullanıcı dostu akademik yönetim paneli.
- **📊 AI Analiz Laboratuvarı:** Makale metadatalarını, okuma sürelerini ve güven skorlarını anlık olarak hesaplayan gelişmiş bir analiz merkezi.

## 🛠️ Teknoloji Yığını (Tech Stack)

| Katman | Kullanılan Teknolojiler |
| :--- | :--- |
| **Frontend** | React, Vite, Framer Motion, Force-Graph, Lucide Icons |
| **Backend** | Node.js, Express.js |
| **Graph DB** | Neo4j (Graph Relationship Mapping) |
| **Document DB** | MongoDB (Content & Metadata Storage) |
| **Styling** | Vanilla CSS (Ultra-Custom Premium UI) |
